# foundry-tesoroygloria
Ficha y sistema de Tesoro y Gloria usando Sandbox (v. 0.16.3) para Foundry VTT (v. 13.351).

Para instalar este mundo se debe descargar este repositorio y pegarlo dentro de \FoundryVTT\Data\worlds.

Una vez intalado e iniciado, se debe dar al botón "Build Actor Templates" para que las plantillas de PJ y PNJ funcionen. Tras esto ya se pueden crear nuevos personajes, pnj y monstruos.
