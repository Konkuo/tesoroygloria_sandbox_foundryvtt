# foundry-tesoroygloria
Ficha y sistema de Tesoro y Gloria usando Sandbox (v. 0.16.3) para Foundry VTT (v. 13).

Para instalar este mundo con la ficha lista descargar este repositorio y pegarlo dentro FoundryVTT\Data\worlds
Una vez intalado e iniciado, se debe dar al botón "rebuild character sheets" para que las fichas plantilla de PJ y PNJ funcionen.
